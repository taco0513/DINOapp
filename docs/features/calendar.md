# Calendar Feature

DINO의 캘린더 기능은 여행 일정을 시각적으로 관리하고 추적할 수 있게 해줍니다.

## 주요 기능

### 월간 보기

- 한 달 단위로 여행 일정 확인
- 각 날짜에 여행 정보 표시
- 셰겐 지역 체류일 하이라이트
- 오늘 날짜 강조 표시

### 연간 보기

- 12개월 전체 여행 현황 한눈에 확인
- 월별 여행 횟수 및 셰겐 체류일수 표시
- 연간 통계 요약
  - 총 여행 횟수
  - 총 셰겐 체류일수
  - 방문 국가 수
  - 총 여행일수

### 상세 정보

- 날짜 클릭 시 해당일 여행 정보 표시
- 여행 기간, 목적, 국가/도시 정보
- 셰겐 지역 여부 표시

## 사용 방법

1. `/calendar` 페이지로 이동
2. 상단 네비게이션으로 월 이동
3. "연간 보기" 버튼으로 뷰 전환
4. 날짜 클릭으로 상세 정보 확인

## 색상 코드

- **파란색 테두리**: 오늘 날짜
- **초록색 배경**: 일반 여행
- **보라색 배경**: 셰겐 지역 여행
- **회색 배경**: 선택된 날짜

## 기능 특징

- 반응형 디자인으로 모바일 최적화
- 부드러운 애니메이션 전환
- 직관적인 네비게이션
- 로컬 스토리지 활용으로 빠른 로딩

# API 문서 페이지 (/docs/api)

## 개요

DINO API의 모든 엔드포인트와 사용법을 상세히 설명하는 개발자 문서 페이지입니다.

## 주요 기능

- REST API 엔드포인트 목록
- 요청/응답 예제
- 인증 방법 안내
- 에러 코드 설명

## API 대분류

- 인증 API
- 여행 관리 API
- 셰겐 계산 API
- 관리자 API
- 모니터링 API

## 기술적 구현

- OpenAPI 3.0 스펙
- Swagger UI 통합
- 상호작용 데모

## 개선 가능 사항

- SDK 생성
- Postman 컴렉션
- GraphQL 지원
